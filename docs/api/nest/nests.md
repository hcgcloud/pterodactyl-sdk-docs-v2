# List Nests <small>with Application API</small>
Get the collection of nests.

## Usage
``` php
<?php
	$pterodactyl->nests(int $page = 1);
?>
```

## Parameters

| Parameter | Description | Rules |
| - | - | - |
| page | The page number | |
